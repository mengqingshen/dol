
## 安装

在安装上，和其它基于 yeoman 的脚手架的安装方式是一致的。

```bash
$ npm i yo -g 
$ npm i generator-dolphin -g
```

### 初始化项目

> 1. 通常，项目是首先在 git 托管平台创建的，让后 clone 到本地，这时项目已经有了顶层目录，因此在项目的顶层目录下，直接 `yo dolphin`， 然后根据提示操作即可。
> 2. 项目初始化完成后，会自动执行 `npm start`
